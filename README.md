# Global-Seismic-rends
Interactive data analysis and visualization of global earthquake trends using Python, MySQL, and Streamlit.
# Project Overview
The goal of this project is to analyze earthquake data and identify trends in magnitude, depth, time, and geographic distribution using an interactive Streamlit dashboard.
# Tools & Technologies
- Python
- MySQL
- Streamlit
- Pandas,
- SQLAlchemy
# Project Features
- Real-time data fetching from USGS Earthquake API
- Data cleaning and preprocessing
- Storage using MySQL database
- Interactive visualizations using Streamlit
- Analysis of earthquakes
# Data Source
- USGS Earthquake API
